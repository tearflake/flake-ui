3.1. making a local copy

F

irst, we need to make sure we have installed a local web server like Apache HTTP Server or similar software. We also need to make sure to have an access to a file manager and a text editor which we will use to access and edit FlakeUI content files.

To install FlakeUI, we have to download the FlakeUI software bundle from dedicated github pages. After downloading the software, we have to unpack the downloaded zip archive to a local server folder. Unpacking the archive will produce the following file-folder structure: 


 !
 +-+ flakeUI-home-folder
   +-+ contents
   ! +-- ...
   !
   +-+ media
   ! +-- ...
   !
   +-+ src
   ! +-- ...
   !
   +-- changelog.md
   +-- index.html
   +-- init.json
   +-- README.md
   +-- LICENSE

Now that we have unpacked the archive, we can test it by entering the local HTTP server address into a web browser (usually localhost). We should be able to see the exact copy of these instructions in the browser.
